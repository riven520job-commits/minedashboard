# MineDashboard - Personal Life Dashboard

## Overview
Notion 風格的個人生活管理 Web App，整合目標追蹤、日語學習、日記、冥想、旅行清單、衣櫥與靈感收藏。

## Features
- Daily goals and habit tracking
- Japanese dictionary and review notes
- Manifestation journal
- Shrine visit records
- Meditation timer and notes
- Wardrobe and outfit generator
- Link and inspiration board
- Cloud sync
- Multi-language support
